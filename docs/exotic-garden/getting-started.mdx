# 入门

## 1. 灌木丛、植物与果树

当你所在的世界启用了异域花园生成，你便可以在世界各地寻找到灌木丛、植物与果树。

如果你找到了灌木丛或植物（它们看起来只有一个树叶方块在地上，或是树叶方块上有对应的产物），只需要右键点击，即可收集产物。

<ImageGallery images={[
  { src: '/images/exotic-garden/plants-and-bushes.png', alt: '树莓灌木丛、番茄植物、香菜植物', sizePercent: 50 }
]} />

果树由树干（原版原木）、树叶（原版树叶）和果实组成。你可以右键点击果实收集它们。

<ImageGallery images={[
  { src: '/images/exotic-garden/tree.png', alt: '椰子树', sizePercent: 50 }
]} />

## 2. 厨房

在收集到各种产物后，你需要建造一个**厨房**。厨房是一个多方块结构。<br />
将厨房摆放好后，你就可以制作食物了。

<ImageGallery images={[
  { src: '/images/exotic-garden/kitchen.png', alt: '厨房', sizePercent: 50 }
]} />

在发射器中放入配方，然后右键点击活板门，即可制作食物。

## 3. 食物

果汁和冰沙是可以放进[小冰柜](https://slimefun-wiki.guizhanss.cn/Cooler)里的。

## 4. 魔法植物

**魔法植物**，它们会产出**魔法精华**。<br />
当你收获足够的魔法精华时，你就可以合成出各种资源。
