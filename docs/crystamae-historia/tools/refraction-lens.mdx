# 折射棱镜

<ImageGallery images={[
  { src: '/images/crystamae-historia/refracting-lens.png', alt: '折射棱镜' }
]} />

折射棱镜用于“查看”方块内部抽取出的魔法水晶在世界之中的流动。

它有以下具体用途：

- *右键点击*[液化池](../spell-crafting/liquefaction)将显示其中包含的魔法水晶的总量和类型
- *右键点击*[经验收集器](../gadgets/experience-collector)将显示当前存储的经验值
