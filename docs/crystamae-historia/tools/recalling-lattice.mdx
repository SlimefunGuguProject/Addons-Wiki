# 魔法传送网

<ImageGallery images={[
  { src: '/images/crystamae-historia/recalling-lattice.png', alt: '魔法传送网' }
]} />

魔法传送网使用魔法锭中留下的潜在水晶之力，形成与特定空间点的链接。<br />
当被使用时，传送网将瞬间将持有者带回该地点。

目的地附近必须有强大的水晶流动。目前已知的是[路标](../gadgets/diverging-waystone)。
