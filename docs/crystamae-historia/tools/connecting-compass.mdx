# 连接指南针

<ImageGallery images={[
  { src: '/images/crystamae-historia/connecting-compass.png', alt: '连接指南针' }
]} />

连接指南针的尖端浸润了一层历史水晶，使其始终记住并指向所选方向。

该效果仅在保存的位置与玩家在统一维度时生效。

## 使用

| 操作 | 效果 |
| --- | --- |
| 右键点击 | 指南针将发射一道粒子光束指向其保存的位置。 |
| Shift + 右键点击 | 指南针将记录玩家当前位置作为新的保存位置。 |
