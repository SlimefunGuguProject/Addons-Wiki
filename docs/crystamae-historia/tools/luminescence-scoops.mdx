# 发光勺

<ImageGallery images={[
  { src: '/images/crystamae-historia/luminescence-scoops.png', alt: '发光勺' }
]} />

发光勺从附近的天体水晶中汲取能量，可以在任意位置生成一个无形的漂浮光源。<br />
如果对已放置的光源使用，则会收回光源。

发光勺有一定的使用次数，但可以通过收回之前放置的光源来重新填充。

## 等级

| 等级 | 使用次数 |
| --- | --- |
| 1级 | 25 |
| 2级 | 75 |
| 3级 | 250 |

## 使用

| 操作 | 效果 |
| --- | --- |
| 右键点击 | 在看向的方块上放置一个光源。 |
| Shift + 右键点击 | 收回面前的光源。 |
