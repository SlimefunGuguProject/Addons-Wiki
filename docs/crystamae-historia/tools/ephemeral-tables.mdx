## 临时工作台

临时工作台可以让你在任何地方访问原版工作台，只需要**右键点击**即可打开。便携的解决方案！

## 临时融合工作台

临时融合工作台是升级版的工作台，可以让你合成原版工作台配方与增强型工作台配方。还是便携的解决方案！
