# 温室玻璃

<ImageGallery images={[
  { src: '/images/crystamae-historia/greenhouse-glass.png', alt: '温室玻璃' }
]} />

温室玻璃可以加速作物的生长。作物必须有足够的光照，且位于温室玻璃下方1格的同高30格范围内。

每粘液刻，受影响的作物有概率生长到下一阶段，直到成熟。

| 类型 | 概率 |
| --- | --- |
| 普通 | 5% |
| 高级 | 10% |
| 巨型 | 20% |
