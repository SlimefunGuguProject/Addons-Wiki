# 防刷怪蜡烛

<ImageGallery images={[
  { src: '/images/crystamae-historia/verache-candles.png', alt: '防刷怪蜡烛' }
]} />

这个魔法道具发出的光线可以清洁周围的区域，并防止任何怪物生成。蜡烛会在一段时间后燃烧殆尽。

| 等级 | 范围 | 持续时间（限时时间） |
| --- | --- | --- |
| 1级 | 16 | 2 小时 |
| 2级 | 32 | 24 小时 |
| 3级 | 64 | 48 小时 |
| 4级 | 64 | 14 天 |
