# 虚空裂隙

<ImageGallery images={[
  { src: '/images/crystamae-historia/voids.png', alt: '虚空裂隙' }
]} />

将虚空、机械和水晶结合，可以撕裂宇宙，生成一个虚空裂隙并吸入附近的物品。

幸运的是，这个裂隙很友好，可以通过货运取出物品。

| 2级 | 7 |
