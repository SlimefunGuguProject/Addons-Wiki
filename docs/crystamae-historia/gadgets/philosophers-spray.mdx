# 哲学家喷雾

<ImageGallery images={[
  { src: '/images/crystamae-historia/philosophers-spray.gif', alt: '哲学家喷雾' }
]} />

哲学家喷雾是对手持[置换器](../tools/displacer)的一种“升级”。

当接收到红石信号时，上方的方块将被置换为另一维度的方块。

关于可置换的方块，查看[这里](../tools/displacer#displacements)。

:::info

哲学家喷雾仅能置换方块，不能置换实体。

:::
