# 灼烧板与陷阱板

<ImageGallery images={[
  { src: '/images/crystamae-historia/damage-plates.png', alt: '灼烧板与陷阱板' }
]} />

将石质压力版与炼金、虚空和机械水晶结合，可以使其每粘液刻对其上方所有生物造成伤害。

将虚空水晶替换为天体水晶，可以得到一个陷阱板，它会应用其分配的药水效果。

| 等级 | 伤害 | 玩家掉落 |
| --- | --- | --- |
| 灼烧板1级 | 1 | 否 |
| 灼烧板2级 | 1 | 是 |
| 灼烧板3级 | 2 | 是 |
| 灼烧板4级 | 3 | 是 |
| 陷阱板1级 | - | 否 |
