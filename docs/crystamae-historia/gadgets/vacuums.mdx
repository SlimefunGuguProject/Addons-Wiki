# 空气炮

<ImageGallery images={[
  { src: '/images/crystamae-historia/vacuums.png', alt: '空气炮' }
]} />

与驱散之灯不同，空气炮使用历史、虚空和哲学水晶，因此可以将任何实体推开。缺点是它只能朝一个方向推动。

| 等级 | 范围 |
| --- | --- |
| 1级 | 5 |
| 2级 | 10 |
| 3级 | 15 |

推动速度均为 1 米/秒。
