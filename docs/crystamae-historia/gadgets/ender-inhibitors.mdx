# 末影人抑制器

<ImageGallery images={[
  { src: '/images/crystamae-historia/ender-inhibitors.png', alt: '末影人抑制器' }
]} />

通过运用末影人的共振来抑制它们，末影人抑制器会阻止它们传送。一旦末影人离开抑制器的范围，该效果在 2 秒后消失。

| 等级 | 范围 |
| --- | --- |
| 1级 | 4 |
| 2级 | 8 |
