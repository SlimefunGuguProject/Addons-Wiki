# 电力炼金厂

电力炼金厂可以将金粉直接冶炼成指定克拉的[金锭](https://slimefun-wiki.guizhanss.cn/Gold-Ingot)。

## 使用

<ImageGallery images={[
  { src: '/images/foxy-machines/gold-refinery-gui.png', alt: '电力炼金厂' }
]} />

打开电力炼金厂的界面，可以在左侧选择冶炼的配方。

然后，连接到货运/网络并输入金粉，数量足够后机器会产出指定克拉的金锭。

## 信息

**电力消耗**: 36 J/s
