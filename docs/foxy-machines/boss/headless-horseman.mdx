# 无头骑士

<ImageGallery images={[
  { src: '/images/foxy-machines/headless-horseman.png', alt: '无头骑士', sizePercent: 50 }
]} />

无头骑士是神秘科技中的首领生物之一。需要制作刷怪蛋来召唤（需要[精灵女王之心](./pixie-queen)）。

本体是骷髅，拥有骷髅的大部分特性。

## 属性

生命值: 1<br />
伤害: 28

免疫任何爆炸、荆棘伤害。

无头骑士每次攻击附带 12% [穿甲伤害](../damage-types)。

## 生成

无头骑士生成时会骑在一只骷髅马上。

在骷髅马上时，无头骑士本体免疫所有伤害。

### 骷髅马属性

:::note 属性

生命值：132<br />
护甲：30<br />
药水效果：抗性提升 5

:::

免疫任何挤压、中毒、爆炸伤害。

> Minecraft Wiki:
> 抗性提升: 将所有受到的伤害（不包括虚空、饥饿与/kill造成的）随等级提升逐渐减免，并在5级时完全免疫。

骷髅马死亡后无头骑士也会一起死亡。

:::tip

由于无头骑士在骷髅马上时是无敌的，所以只能攻击骷髅马。

然而，由于抗性提升5的药水效果，骷髅马免疫任何原版武器的攻击。

因此，你需要拥有穿甲伤害的武器来攻击骷髅马。

:::

## 技能

### 远程攻击

无头骑士会向玩家射出箭矢，造成 22 点伤害。

### 引雷

无头骑士在玩家的位置召唤闪电，造成 12.4 的闪电伤害 + 1.72 的真实伤害。

### 召唤

无头骑士召唤出3只**地狱犬**来协助战斗。

<ImageGallery images={[
  { src: '/images/foxy-machines/helldog.png', alt: '地狱犬', sizePercent: 50 }
]} />

地狱犬本体为狼，拥有狼的大部分特性，生成时身上会着火。

:::note 属性

生命值：30<br />
伤害：26<br />
药水效果：防火

:::

地狱犬每次攻击附带 20% [穿甲伤害](../damage-types)。

## 掉落物

无头骑士死亡后会掉落**邪恶南瓜**，以及 2000-2800 点经验。
