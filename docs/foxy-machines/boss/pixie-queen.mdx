# 精灵女王

<ImageGallery images={[
  { src: '/images/foxy-machines/pixie-queen.png', alt: '精灵女王' }
]} />

精灵女王是神秘科技中的首领生物之一。需要制作刷怪蛋来召唤。

本体是恼鬼，拥有恼鬼的大部分特性。

:::note 属性

生命值：800<br />
伤害：28

:::

精灵女王每次攻击附带 12% [穿甲伤害](../damage-types)。

## 技能

### 远程攻击

精灵女王会向玩家射出箭矢，造成 24 点伤害。

### 召唤

精灵女王会召唤出 3-4 只**精灵**来协助战斗。

<ImageGallery images={[
  { src: '/images/foxy-machines/pixie.png', alt: '精灵' }
]} />

:::note 属性

生命值：5<br />
伤害：24

:::

精灵每次攻击附带 20% [穿甲伤害](../damage-types)。

## 掉落

精灵女王死亡后会掉落**精灵女王之心**，以及 1400-2000 点经验。
