# 智能工厂

智能工厂是蓬松机器提供的终极懒人机器，可以直接使用矿粉和煤炭制作成指定的物品。

## 界面

右键点击智能工厂可以打开以下界面：

<ImageGallery images={[
  { src: '/images/fluffy-machines/smart-factory.png', alt: '智能工厂' }
]} />

左侧输入栏分成了两个区域：

- **煤炭区**：第一行，该区域将接收来自货运网络的煤炭，共3格，最多可存放3组煤炭。
- **矿粉区**：剩余三行，该区域将接收来自货运网络的其他物品，共9格，最多可存放9种矿粉。

:::info 注意

对于**矿粉区**，货运只会向现有的物品补充数量，不会填充空的格子。

:::

在右下角可以看到当前设定的配方、所需的材料以及制作进度。

## 使用

手持支持制作的物品（可在粘液科技指南书中查看），*Shift + 右键点击*智能工厂即可设置配方。

智能工厂将在原材料充足时开始制作指定的物品。

## 等级

目前智能工厂有1个等级。

**速度**: 1x<br />
**电力消耗**: 2048 J/s<br />
**可储存**: 3072 J
