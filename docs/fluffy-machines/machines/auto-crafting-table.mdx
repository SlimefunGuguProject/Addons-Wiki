# 全自动工作台（原版）

全自动工作台（原版）是蓬松机器中的自动合成机器，可以使用原版工作台的配方来合成物品。

## 使用

打开全自动工作台（原版）可以看到以下界面：

<ImageGallery images={[
  { src: '/images/fluffy-machines/auto-crafting-table.png', alt: '全自动工作台（原版）' }
]} />

左侧蓝色区域是摆放配方的地方，而右侧黄色区域是摆放目标物品的地方。

使用时，需要在黄色区域摆放需要合成的物品，然后在蓝色区域摆放对应的配方。

点击上方的开关即可开始自动合成。

## 信息

**速度**: 1x<br />
**电力消耗**: 256 J/s<br />
**可储存**: 384 J
