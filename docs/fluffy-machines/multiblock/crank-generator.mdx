# 手摇发电机

手摇发电机是一个多方块结构，可以让玩家挥动双手进行发电。

## 建造

放置发电机芯，确保发电机芯接入了电力网络。<br />
然后，在上方放置一个拉杆。

<ImageGallery images={[
  { src: '/images/fluffy-machines/crank-generator.png', alt: '手摇发电机' }
]} />

## 使用

右键点击拉杆即可进行一次发电。

**发电效率**: 16J 每次使用<br />
**可储存**: 64 J
