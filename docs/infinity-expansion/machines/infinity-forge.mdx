# 无尽工作台

无尽工作台是用于制造**无尽物品**分类中的物品所需的工作台。

## 界面

打开无尽工作台可以看到以下界面：

<ImageGallery images={[
  { src: '/images/infinity-expansion/infinity-forge.png', alt: '无尽工作台界面' }
]} />

其中左侧的 6x6 区域为配方摆放区域。

点击右上角的书本图标可以查看所有无尽物品的配方，也可以通过 Slimefun 指南书查看。

当摆放好配方后，点击合成（绿色玻璃板）即可合成出一件无尽物品。物品将在输出栏（橙色玻璃板背景）中。

## 信息

**电力消耗**：10,000,000 J 每次使用
