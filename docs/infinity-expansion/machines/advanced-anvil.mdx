# 高级铁砧

高级铁砧用于合并两个物品的附魔。

  { src: '/images/infinity-expansion/advanced-anvil.png', alt: '高级铁砧' }

## 使用

在输入栏位中放入2个物品。必须满足以下条件：

- 两个物品必须是同类型的，或第二个物品为附魔书。
- 第一个物品必须允许附魔（部分粘液科技物品禁止附魔）。
- 第二个物品必须允许祛魔（部分粘液科技物品禁止祛魔）。
- 两个物品必须都有可用附魔（必须是能从 Bukkit 注册中心获取的附魔，包括原版附魔与部分插件添加的附魔）。
- 当两个物品有同一种附魔类型时，右侧物品的附魔等级必须大于等于左侧物品的附魔等级。

如果上述条件满足，将会在下方显示合并后的物品，点击该物品即可合并。

- 左侧物品会作为附魔合并后的基础物品，右侧物品会消失。
- 相同类型的附魔，如果等级一致，则合并后的附魔提升一级。否则，取等级更高的作为最终等级。
- 右侧物品包含左侧物品没有的附魔时，会将这些附魔移动到左侧物品。

## 等级上限

合并后的附魔不会超过所设置的等级上限。这些上限可通过无尽贪婪的配置文件更改。

**默认**上限如下：

- 锋利：9
- 亡灵杀手：9
- 节肢杀手：9
- 保护：10
- 时运：6
- 效率：7
- 火焰附加：6
- 力量：7
- 荆棘：13
- 抢夺：6
- 耐久：13
- 其他附魔与原版上限一致。

## 信息

**电力消耗**：100,000 J 每次使用
