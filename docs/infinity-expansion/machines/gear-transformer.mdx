# 装备材质转换器

装备材质转换器是可以更换装备材质的机器。

## 使用

在左侧放入要转换材质的装备，在右侧放入转换材质所需的物品。

下方表格列出了转换到指定材质所需的物品：

<Tabs>
  <TabItem value="tools" label="工具" default>
| 材质 | 所需物品 |
| --- | --- |
| 木 | 橡木原木 * 4 |
| 石 | 圆石 * 4 |
| 铁 | 铁锭 * 4 |
| 金 | 金锭 * 4 |
| 钻石 | 钻石 * 4 |
| 下界合金 | 下界合金锭 * 2 |
  </TabItem>
  <TabItem value="armor" label="防具">
| 材质 | 所需物品 |
| --- | --- |
| 皮革 | 皮革 * 9 |
| 锁链 | 锁链 * 9 |
| 铁 | 铁锭 * 9 |
| 金 | 金锭 * 9 |
| 钻石 | 钻石 * 9 |
| 下界合金 | 下界合金锭 * 2 |
  </TabItem>
</Tabs>

## 配置

可以在配置文件中更改是否允许转换粘液科技装备的材质 (`balance-options.allow-sf-item-transform`)，默认关闭。

## 信息

**电力消耗**: 12,000 J 每次使用
