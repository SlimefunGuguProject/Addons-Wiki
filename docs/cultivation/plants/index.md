# 植物

农耕工艺中有两种植物：原始植物与农耕植物。

原始植物是基础植物，仅用于培育，不生成任何物品。

农耕植物不仅可以用于[培育](../overview/crop-breeding)，还可以生成物品。

交叉培育可以产生新的类型的植物。

所有非原始植物、财富与光谱都需要通过交叉培育来获得。
