# 培育作物

<ImageGallery images={[
  { src: '/images/cultivation/crop-breeding.png', alt: '培育作物' }
]} />

培育作物是用两株植物来培育（简单培育）或产生新的种类的植物（交叉培育）。

要培育作物，你需要两株植物在同一高度，同一方向上放置，中间间隔1格空气。

如果两株作物中间出现黑色颗粒，这意味着这两株植物无法进行交叉培育。在培育时，要留意这些黑色颗粒。

不是所有的作物都可以与其他作物交叉培育。

如果两株相同的植物用于培育，则子植物将与父母植物相同。
