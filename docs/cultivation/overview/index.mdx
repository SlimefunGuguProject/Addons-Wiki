# 总览

什么是农耕工艺？

农耕工艺是一个农作物种植插件，添加了大量的独特植物，具有自己的特点！<br />
你可以通过不同方式处理材料，从而制作各种食物。

该附属由 Sefiraat 和 Jeff 开发，且社区中的许多人通过提交 PR 添加了更多的食物，做出了贡献。
