# 作物架

作物架是一种种植工具，用于种植农作物。右键点击农耕工艺作物即可放置作物架。

放置一个作物架可以让作物生长，放置两个作物架可以允许作物进行培育。

使用作物架后，植物最终会长成植物形态（这可能需要一些时间）。
