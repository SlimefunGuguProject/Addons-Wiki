# 厨房柜台

厨房柜台可以让你从原材料准备食材与制作食物。

要使用柜台，手持材料*右键点击*柜台。

手持任意方块*右键点击*柜台可以更改柜台的下半部分。这不会消耗你用来定制的方块。<br />
*Shift+右键点击*柜台则可以更改上半部分。
