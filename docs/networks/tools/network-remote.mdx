# 网络远程访问器

<ImageGallery images={[
  { src: '/images/networks/network-remote.png', alt: '网络远程访问器' }
]} />

网络远程访问器允许你远程访问[网格](../network-nodes/network-grid)。

手持一个远程访问器*Shift + 右键点击*一个网格来绑定。

如果在范围内，手持绑定好的远程访问器即可打开绑定的网格。

绑定的网格所在的区块需要加载才能使用远程访问，所以你需要确保网格满足以下条件之一：

- 在出生点区块
- 所在区块已加载
- 使用区块加载器保持区块加载
