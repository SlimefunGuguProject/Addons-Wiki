# 网络绘制器

<ImageGallery images={[
  { src: '/images/networks/network-crayon.png', alt: '网络绘制器' }
]} />

网络绘制器可以切换网络绘制。手持网络绘制器右键点击[网络控制器](../network-nodes/network-controller)即可切换。

当网络绘制开启时，网络中所有可以显示粒子的方块会在特定操作时显示粒子。
