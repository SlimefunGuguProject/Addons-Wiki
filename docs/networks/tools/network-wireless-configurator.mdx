# 网络无线配置器

<ImageGallery images={[
  { src: '/images/networks/network-configurator.png', alt: '网络无线配置器' }
]} />

网络无线配置器可以保存一个[网络无线接收器](../network-nodes/network-wireless-receiver)的位置，并将其应用至[网络无线发射器](../network-nodes/network-wireless-transmitter)。

*右键点击*一个网络无线接收器可以储存其位置。

*Shift+右键点击*一个网络无线发射器以应用配置，完成无线传输的配置。
