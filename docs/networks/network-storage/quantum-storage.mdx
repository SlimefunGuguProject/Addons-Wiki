# 量子存储

量子存储是网络提供的存储解决方案，可以存储大量的单一物品，可接入网络。

## 使用

将量子存储摆放出来，右键点击即可打开，界面如下图所示：

<ImageGallery images={[
  { src: '/images/networks/quantum-storage.png', alt: '量子存储界面' }
]} />

### 设置物品

在使用之前，需要设置量子存储所存储的物品。拿起需要存储的物品，然后点击下方黄绿色玻璃板即可设置。

### 存入/取出

在左侧可以手动存入物品，在右侧可以手动取出物品。

可以接入货运，但**不推荐**。

推荐使用[网络监视器](../network-nodes/network-monitor)，将量子存储接入网络。

### 满载清空输入

你可以设置满载时清空输入来避免货运堵塞。

该设置默认开启。<br />
*Shift+右键点击*下方的黄绿色玻璃板来进行切换。

## 等级

不同等级的量子存储有不同的容量。

| 量子存储 | 存储容量 |
| --- | ------- |
| 4k | 4,096 |
| 32k | 32,768 |
| 262k | 262,144 |
| 2m | 2,097,152 |
| 16m | 16,777,216 |
| 134m | 134,217,728 |
| 1b | 1,073,741,824 |
| ∞ | 2,147,483,647 |
