# 网络推送器

<ImageGallery images={[
  { src: '/images/networks/network-pusher.png', alt: '网络推送器' }
]} />

网络推送器可以将网络中指定的物品送入相邻的 Slimefun 方块中。

网络推送器是一个带有方向选择的网络方块。目标方块必须是 Slimefun 方块，且有输入栏可以通过**货运输入**物品。<br />
你可以用[网络配置器](../tools/network-configurator)来复制方向设置。

右键点击网络推送器可以打开交互界面，允许你选择方向并在右侧放入物品作为模版。

目标方块必须有可用的输入空间让推送器工作。推送器会按照[取出机制](../basics/mechanism)从网络中取出对应的物品。

如果网络启用了[绘制](../tools/network-crayon)，在推送器推送一个物品时，会在方块上方显示粒子效果，标记推送方向。
