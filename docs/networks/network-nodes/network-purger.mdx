# 网络清除器

<ImageGallery images={[
  { src: '/images/networks/network-purger.png', alt: '网络清除器' }
]} />

网络清除器用于清除网络中指定的物品。

右键点击网络清除器可以打开交互界面，设置模版物品。每粘液刻会从网络中移除一组对应的物品。

如果网络启用了[绘制](../tools/network-crayon)，在清除一组物品时，会在方块上方显示粒子效果。
