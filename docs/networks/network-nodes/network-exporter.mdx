# 网络出口

<ImageGallery images={[
  { src: '/images/networks/network-exporter.png', alt: '网络出口' }
]} />

网络出口可以让玩家可以从网络中**取出**物品，以供货运网络使用。

网络出口需要放入一个物品作为模版。每粘液刻，网络会根据[取出机制](../basics/mechanism)从网络中取出一组对应的物品。货运系统可从中取出物品。

:::info

不同于货运节点（输出），网络出口不会向相邻方块输出物品。需要使用货运或手动取出物品。

:::
