# 网络配方编码器

<ImageGallery images={[
  { src: '/images/networks/network-encoder.png', alt: '网络配方编码器' }
]} />

网络配方编码器可以将配方写入一张空白的蓝图中。

<ImageGallery images={[
  { src: '/images/networks/network-encoder-gui.png', alt: '网络配方编码器界面' }
]} />

要编码配方，首先需要将一张空白蓝图放入左侧的蓝图栏（蓝色背景）。<br />
然后，在中间区域放入需要编码的配方物品。<br />
最后，点击右上角的编码按钮，开始进行编码。


:::warning

编码操作会消耗配方栏的所有物品，无法返还。

:::
