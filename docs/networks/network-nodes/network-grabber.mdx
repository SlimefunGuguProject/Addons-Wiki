# 网络抓取器

<ImageGallery images={[
  { src: '/images/networks/network-grabber.png', alt: '网络抓取器' }
]} />

网络抓取器可以从相邻的 Slimefun 方块中提取物品，并存入网络中。

网络抓取器是一个带有方向选择的网络方块。目标方块必须是 Slimefun 方块，且有输出栏可以通过**货运取出**物品。<br />
你可以用[网络配置器](../tools/network-configurator)来复制方向设置。

网络中需要有可用空间来让抓取器工作。抓取器会按照[存储机制](../basics/mechanism)将物品存入网络中。

如果网络启用了[绘制](../tools/network-crayon)，在抓取器获取一个物品时，会在方块上方显示粒子效果，标记抓取方向。
