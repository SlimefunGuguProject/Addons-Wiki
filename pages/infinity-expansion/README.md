# 主页

InfinityExpansion 是一个包含大量游戏后期物品的 Slimefun4 附属插件。  
你可以通过本 Wiki 来了解该附属插件的玩法。本 Wiki 由[ybw0014](https://github.com/ybw0014)维护。

本 Wiki 将使用 InfinityExpansion 的非官方中文名称**无尽贪婪**来指代 InfinityExpansion。
