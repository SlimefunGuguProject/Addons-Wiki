WilderNether 是2022年粘液科技附属竞赛参赛作品，作者为 SchnTgaiSpock。

粘液科技简中汉化组对该附属进行了汉化，并确定了非官方中文译名: 迷狱生机。  
[点击此处](https://builds.guizhanss.net/SlimefunGuguProject/Wildernether/master)下载迷狱生机汉化版。
