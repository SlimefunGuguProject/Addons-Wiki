# 主页

SlimeCustomizer 是一个 Slimefun 附属，可以让服主只通过配置文件就能创建一个附属，操作简单易懂。

其非官方中文名为自定义粘液附属。[点击此处](/Install#SlimeCustomizer)下载自定义粘液附属汉化版。
