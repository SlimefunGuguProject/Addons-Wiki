## 什么是至尊研究院? :id=what-is-supreme

至尊研究院（Supreme 2.0）由 RelativoBR 等人开发，是粘液科技插件的一个附属插件。

它向粘液科技添加了非常多的机器和装备

Minecraft 版本：1.17 及以上

对工业分支要求：非常高  

对魔法分支要求：较低 ~ 高  

对生物群系和结构的要求：低
